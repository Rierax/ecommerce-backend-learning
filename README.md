Шаг 1.
Запусти команду:

Windows:
```mvnw.cmd clean install```

Windows (PowerShell):
```.\mvnw.cmd clean install```

Linux/macOS:
 ```./mvnw clean install```

Шаг 2.
Запусти тесты:

 Windows:
 ```mvnw.cmd test```

 Windows (PowerShell):
 ```.\mvnw.cmd test```

 Linux/macOS:
  ```./mvnw test```

Если не выполняется команда, убедись в том что у тебя установлена Java SDK, проверьить это можно, выполнив команду:
```java --version```
